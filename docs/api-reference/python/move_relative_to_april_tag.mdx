---
title: move_relative_to_april_tag
---

Move the robot relative to a detected AprilTag.

## Parameters

- `tag_id`: ID of the AprilTag to move relative to.
- `offset`: Dictionary containing x, y, z, roll, pitch, yaw offsets.

## Example Usage

```python
offset = {"x": 10, "y": 0, "z": 0, "roll": 0, "pitch": 0, "yaw": 0}
await client.move_relative_to_april_tag(1, offset)
```
